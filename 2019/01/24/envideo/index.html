Charles一分钟APP抓包教程

发表于 2019-01-24 | 更新于 2019-08-12 | 分类于 工具 | 阅读次数：
本文字数： 508 | 阅读时长 ≈ 1 分钟

今天我正在灰常认真的上班工作，忽然“叮咚”一响，微信来消息了，原来是婷小妹给我发了个消息

原来是儿子早中心购买的英文课视频一年期满后就不能观看了，除非要购买他们一套教学玩具才赠送永久观看权限，三四百块呀，我下意识的摸了摸钱包，干瘪干瘪的，只好默默的打开 Charles

打开后在菜单中找到 Proxy 把 macOS Proxy 前的勾去掉，因为现在是 APP 抓包

然后 Proxy –> Proxy Settings… 打开设置，端口填 8888

接下来就是手机端的设置了，PC 和手机必须要在同一 wifi 网络内。

进入你手机当前连接的 wifi 可以看到有个 http 代理，选择手动，服务器填你 PC 的的 ip 地址，这个你自己应该会看啦，略过，端口就填 Charles 中设置的 8888，选择存储。

保存后你会看到 PC 端会弹出提示，选择 Allow 就可以了

接下来就是打开你要抓包的APP就可以看到请求啦，如果数据没有加密一般都可以看得到，下面就是我要的课程视频链接啦

可以交货了 :-)

对了，最后抓完之后记得在手机把 http 代理关闭，要不手机上网会不正常的哦。
